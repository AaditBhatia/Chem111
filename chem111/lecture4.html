  • Significant Figures

  • Rules:
    1. ALl non-zero digits and zeroes between non-zero digits are always significant. Example 1005 kg.
    2. zeroes at beggining of a number are never significant. Ex. 0.02 g
    3. Zeroes are significant if ....
    0.0067 has 2 significant figures
  • Round:
    1. 12.696- 12.7
    2. 16.349- 16.3
    3. 18.351- 18.4. When we are talking about 5, we round to an even number
  • When adding or subtracting values, the answer should have same number of decimal places as the value that has the fewest decimal placed
    If answer contains more sig figures, it must be rounded off.
  • When multiplying or dividing- same number of sig figs as the value with fewest sig-figs
    When answer contains more than correct number of sig-figs- round off.
  • Precision: How close a set of measurement are to each other. THe closer they are the more precise they are
    Accuracy:How close a measurement is to reality/real values
